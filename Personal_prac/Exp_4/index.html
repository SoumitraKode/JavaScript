Experiment 4: String Comparison

String Comparison Types
  • Strict Equality: Returns whether both strings are identical.
  • Length: States which string (if any) is longer, or if they are equal in length.
  • Alphabetical Order: Determines dictionary order using locale comparison.
  • Substring: Indicates if one string appears within the other.
